DK TEX site package

Upload these files to the root of your Vercel/static site project.

Required:
- index.html
- favicon.ico
- favicon.png
- apple-touch-icon.png
- og-image.png

After deployment, check:
https://dktex.lv/favicon.ico
https://dktex.lv/favicon.png
https://dktex.lv/og-image.png

If the favicon in the browser tab does not update immediately, check in Incognito mode or clear browser cache.
